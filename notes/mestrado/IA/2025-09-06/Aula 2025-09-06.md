F[x] é a propagação do literal x na fórmula F
$F = x1 \wedge (\neg x2\vee x3)$
$F[\neg x2]=x1$

## DPLL(F)
F = simplifica(F)
if (F == 0) retorna UNSAT
if (F == 1) retorna SAT
x = escolhe_variável(F)
F' = BCP(F, $x$) // $F'= F[x]$
if (DPLL(F') == SAT) retorna
F" = BCP(F, $\neg x$) // $F" = F[\neg x]$
if (DPLL(F") == SAT) retorna SAT
retorna UNSAT
